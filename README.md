# Suraj Ventures

Suraj Ventures is a feature-first product discovery platform that helps users explore products based on ingredients, benefits, and features rather than brand names.

## Features

- User Authentication
- Product Discovery
- Category-Based Browsing
- Ingredient Search
- Product Details
- Save Products
- Inquiry System
- User Dashboard
- Admin Dashboard
- Responsive Design

## Tech Stack

- React
- Vite
- Tailwind CSS
- Firebase Authentication
- Cloud Firestore
- React Router
